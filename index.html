Manifesto

is an HTML5 Offline Application Cache verification bookmarklet

Manifesto provides a quick and easy way to make sure your HTML5 manifest file is valid and working.

One of the hardest parts of the Offline Application Cache is that when it doesn't work, it fails silently, and you may not even realize it has. This bookmarklet provides a simple way to make sure your page is using the manifest, and if it's not, it helps you diagnose the problems.
